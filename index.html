七个问题

  1. *他的action是什麼？有幾種？ *

    the action space is the set of color adjustment operations A

    行动action是颜色调整操作的集合，共12种

  2. 他的state是什麼？next state是什麼？

    We represent an image I(t) at a step t using a contextual feature F context (I(t)) and a global color feature F color (I(t)).

    the state S is a combination of the contextual feature and the color feature (F context ,F color )

    第T步的图像可以表示为上下文和整合颜色特征，状态s为两者的结合。

  3. 網路怎麼進行學習？下個世代的ground-truth目標是什麼？

    agent网络的优化算法是Adam，目标是12个action对应的值，计算估计值，选择最高的为正的action的值，后对image操作完，求和target图片的二范数距离，和上一步的距离相减作为R值，用于训练网络

  4. 網路架構是什麼？輸入什麼？輸出是什麼？

    全部的算法架构见Fig2 ，agent网络由四个全连接层组成，每层的输出是4096,4096,512,12，所以最后的输出对应12个action的选择，输入就是state

  5. done怎麼決定？也就是停止條件是什麼？

    The agent then chooses an action that has the highest action value and applies the action to the input image if the value is positive. The agent repeats this process and stops when all estimated action values are negative.

    算法选择最高的action作为下一步的操作，当所有的action的值为负时停止

  6. 訓練的時候，一張圖要跑幾次？測試的時候呢？跑幾次？

    最大步数是3000000，测试和训练步数一致，（不太确定。。。）

  7. *測試的時候，是否每次run結果都不一樣？也就是每次的預測結果都不同？ *

    因为有epsilon值，所以应该是会有随机结果产生的。。。。还不是很明白，此外ep值是由步数决定

Detail-enhanced Multi-scale Exposure Fusion in YUV Color Space

Posted on 2019-07-19

【论文笔记】Detail-enhanced Multi-scale Exposure Fusion in YUV Color Space

在YUV颜色空间中的细节提取的多尺度曝光融合

github代码

摘要

利用边缘保持平滑技术对现有的多尺度曝光融合算法进行了改进。然而，基于边缘保持平滑的多尺度曝光融合是移动设备面临的一个难题。本文在YUV颜色空间中设计了一种简单的多尺度曝光融合算法。该算法既能在高动态范围(HDR)场景最亮和最暗的区域保留细节，又能在融合图像中避免颜色失真的同时，实现基于边缘保持平滑的多尺度曝光融合算法。该算法的复杂度约为基于边缘保持平滑的多尺度曝光融合算法的一半。因此，该算法比基于边缘保持平滑的多尺度曝光融合算法更适合智能手机。此外，提出了一种简单的细节增强组件来增强融合图像的精细细节。实验结果表明，所提出的组件可以产生具有明显增强的精细细节和较高的MEF-SSIM值的增强图像。这对于现有的细节增强组件是不可能的。显然，该组件对基于PC的应用程序很有吸引力。

I. INTRODUCTION

主要讲这几件事

  1. 多尺度曝光融合优于单尺度（保持对比度），本文方法是基于Single-scale fusion: An effective approach to merging images这篇文章的算法（这篇是单尺度，效果很好）
  2. 在YUV中实现更佳，因为RGB空间中RGB三个通道相关性太高
  3. 细节提取的方法很好，但是本文要弄个简化的低复杂度的

总得来说的就是性能不差，算法更简

II. MULTI - SCALE EXPOSURE FUSION IN YUV COLOR SPACE

A. Construction of weighting map

在YUV空间构造权重图

采用对比度C、饱和度S和曝光E三种测量方法来计算LDR图像的加权图。

B. Multi-scale exposure fusion

用金字塔图像融合的方法来进行多尺度融合，其中最高一层金字塔权重图使用高斯滤波平滑，原图使用拉普拉斯金字塔第一层。其它层金字塔则是权重图乘原图。

C. ROI exposure fusion

自己设定亮图和暗图的权重

III. DETAIL ENHANCEMENT OF EXPOSURE FUSION IMAGE

构建向量场，然后提取特征，最后特征的指数与图像相乘得到细节提升的融合结果。

从单帧图像中检测和分离烟雾

摘要

文章提出一种从单帧图像中检测和分离烟雾的新方法，具体地说，在大气散射模型的基础上，推导了一种成像模型。将框架分离为拟烟分量和拟背景分量，并将其表示为凸优化，分别利用烟雾分量和背景分量的对偶字典求解稀疏表示问题。提出了一种基于稀疏系数级联的特征检测方法。此外，提出了一种基于抠图的方法，将真实烟雾和背景成分从烟雾检测结果中分离出来。大量的检测实验表明，提出的特征明显优于现有的烟雾检测特征。特别地，该方法能够在灰度框架中区分具有相似视觉外观的其他具有挑战性的对象(如雾/霾、云等)。烟气分离实验也证明了该方法的有效性。
